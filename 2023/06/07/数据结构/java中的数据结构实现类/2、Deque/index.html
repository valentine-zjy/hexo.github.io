2、Deque

Deque（双端队列）

Deque（Double Ended Queue，双端队列）是Java集合框架中的接口，代表一个可以在两端进行插入和删除操作的队列。Deque接口扩展了Queue接口，并提供了在队列的两端进行元素操作的能力。

实现类

Java中提供了多个实现Deque接口的类，常见的有以下几个：

  1. ArrayDeque: 基于可调整大小的数组实现的双端队列。它提供了高效的插入和删除操作，并且没有容量限制。由于其基于数组的实现方式，可以快速随机访问元素。ArrayDeque不是线程安全的，不支持同步访问。

  2. LinkedList: 基于双向链表实现的双端队列。它提供了高效的插入和删除操作，但访问操作的性能相对较慢。LinkedList可以用作双端队列、栈或队列的实现。它不是线程安全的，不支持同步访问。

ArrayDeque

ArrayDeque类是 双端队列的线性实现类。

具有以下特征：

☞ ArrayDeque是采用数组方式实现的双端队列。
☞ ArrayDeque的出队入队是通过头尾指针循环，利用数组实现的。
☞ ArrayDeque容量不足时是会扩容的，每次扩容容量增加一倍。
☞ ArrayDeque可以直接作为栈使用。当用作栈时，性能优于Stack，当用于队列时，性能优于LinkedList。
☞ 无容量大小限制，容量按需增长。
☞ 非线程安全队列，无同步策略，不支持多线程安全访问。
☞ 具有fail-fast特性，不能存储null值，支持双向迭代器遍历。

Deque接口有多个实现类，常用的是ArrayDeque和LinkedList。ArrayDeque基于可调整大小的数组实现，具有常数时间复杂度的大多数操作。LinkedList基于双向链表实现，插入和删除操作较快，但访问操作相对较慢。

使用Deque接口及其实现类，可以实现双端队列的功能，并且能够高效地在队列的两端进行元素操作，例如将元素添加到开头或末尾、获取开头或末尾的元素，以及删除开头或末尾的元素等。
